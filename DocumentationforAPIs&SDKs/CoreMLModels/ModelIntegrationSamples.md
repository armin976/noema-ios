# Model Integration Samples

> Learn how to integrate Core ML models into your app.

## Overview

This section provides code examples and guidance on integrating Core ML models into your app. These samples demonstrate various techniques for loading, compiling, and using models for predictions.

## Samples

- [Downloading and Compiling a Model on the User’s Device](/documentation/coreml/downloading-and-compiling-a-model-on-the-user-s-device) - Install Core ML models on the user’s device dynamically at runtime.
- [MLModelAsset](/documentation/coreml/mlmodelasset) - A wrapper for a Core ML model that can be loaded from a file.
- [MLArrayBatchProvider](/documentation/coreml/mlarraybatchprovider) - A convenience wrapper for an array of feature providers.
- [MLDictionaryFeatureProvider](/documentation/coreml/mldictionaryfeatureprovider) - A convenience wrapper for a dictionary of feature values.
- [MLBatchProvider](/documentation/coreml/mlbatchprovider) - A convenience wrapper for an array of feature providers.
- [MLFeatureValue](/documentation/coreml/mlfeaturevalue) - A wrapper for a single feature value.
- [MLSendableFeatureValue](/documentation/coreml/mlsendablefeaturevalue) - A sendable wrapper for a feature value.
